# Anomaly Report: Day 001

## The Anomaly to Test
**Type Mismatch:** What does the Go compiler do if you try to treat money (an integer) like a name (a string)?

## Execution Steps
1. Write code that intentionally tries to "Add" a string word to your integer balance.
2. Note the specific error message provided by the Go compiler.

## The Fintech Lesson
In a bank, why is it better for a program to **refuse to run** (Compile Error) than to run and produce a "guess" at the math?
