# Altradits — Web

Next.js (App Router) frontend for Altradits.

This app is part of a larger stack (Go API + PostgreSQL + Redis) and is not
meant to be run on its own. For the full setup and startup procedure, see the
[root README](../../README.md#quick-start-after-clone).

Quick reference, once the rest of the stack is running (`make dev-db`,
`make migrate-up`, `make dev-backend`):

```bash
make dev-frontend
# or, from this directory: npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
