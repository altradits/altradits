# Altradits Ecosystem — Beyond Banking

## The Full Vision

Altradits is not just an investment platform. It is a **complete Bitcoin ecosystem** for Kenya:

1. **Investment Pool** — Lock sats for 5+ years, earn monthly profit
2. **Events & Hackathons** — Learn, build, earn sats, get certified
3. **Travel & Leisure** — Book safaris with Gorilla Sats using sats
4. **Crowdfunding** — Well-wishers support students, events, and builders

---

## Module 1: Investment Pool (Core)

| Feature | Description |
|---------|-------------|
| Lock sats | Minimum 5 years. Add anytime. |
| Monthly profit | Credited automatically from low-risk assets |
| After maturity | Choose daily/weekly/monthly/annual profit access |
| Capital preservation | Original sats never leave |

---

## Module 2: Events & Hackathons

### For Event Organizers

| Feature | Description |
|---------|-------------|
| Create event | Title, date, venue, capacity, ticket price (sats) |
| Manage students | View registrations, send materials (PPTs, links, notes) |
| QR check-in | Daily attendance scanning at entry doors |
| Reward sats | Auto-reward students for attending |
| Communications | Built-in chat for before/during/after event |
| Pre-event game | Quiz based on content (Bitcoin, facilitators, venue) |

### For Students (Attendees)

| Feature | Description |
|---------|-------------|
| Register for event | Pay ticket in sats |
| Pre-event game | Play quiz before event starts → earn sats |
| Daily check-in | Scan QR at door → mark attendance |
| Receive materials | PPTs, notes, links in dashboard |
| Submit projects | Homework, hackathon projects |
| Get reviewed | Community members review submissions |
| Earn points | Points from attendance, game, submissions |
| Certificate | Issued after successful completion (daily logins) |

### For Community Reviewers

| Feature | Description |
|---------|-------------|
| Browse submissions | View projects from students |
| Review and rate | Award points |
| Collaborate | Invite students to collaborate based on bio and work |
